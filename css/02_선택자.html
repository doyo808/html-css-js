# CSS 선택자 (Selector)

CSS 선택자 연습하기
  • 태그이름: 해당 태그이름을 가진 모든 요소를 선택
  • #아이디: 해당 ID속성 값을 지닌 요소를 선택 (하나만 선택)
  • .클래스: 해당 클래스 속성 값을 지닌 모든 요소를 선택 (여러개 선택)
  • * : 모든 요소 선택
  • , : 여러개의 선택자를 사용
  • > : 왼쪽 요소의 자식 요소인 오른쪽 요소들을 선택
  • + : 왼쪽 요소의 바로 다음에 오는 형제 요소를 선택
  • ~ : 왼쪽 요소의 다음에 오는 형제 요소들을 모두 선택
  • :first-child : 해당 요소의 첫번째 자식 요소를 선택
  • :last-child : 해당 요소의 마지막 자식 요소를 선택
  • :nth-child(n) : 해당 요소의 n번째 자식 요소를 선택
  • 공백 : 바로 아래에 있는 모든 하위(자손) 요소를 선택
