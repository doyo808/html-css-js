# margin

  • 해당 요소와 다른 요소 간의 거리를 결정하는 여백
  • border 기준 바깥쪽에 존재하는 여백
  • margin-bottion과 margin-top이 만나는 곳에서는 두 값 중 큰 값이 적용된다(합산 x)
  • 좌우의 margin을 auto로 설정하면 해당 요소를 가운데로 정렬해준다

# HTML Box Model

  • HTML은 모든 요소를 박스 형태로 구성한다
  • 제일 안쪽에 centent-box가 있고, content-box를 둘러싼 padding, border, margin의 요소의 크기와 위치를 결정한다
