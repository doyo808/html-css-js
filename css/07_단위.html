# 절대 길이 단위

  • 절대 길이 단위는 고정된 크기를 가지며, 화면 크기와 무관하게 동일한 크기를 유지한다
  • 주로 인쇄물이나 고정된 크기의 요소에 사용된다
  • px: 픽셀 단위, 화면의 픽셀 수를 기준으로 함
  • pt: 포인트 단위, 1pt는 1/72인치
  • cm: 센티미터 단위
  • mm: 밀리미터 단위
  • in: 인치 단위, 1in은 2.54cm

# 상대 길이 단위

  • 상대 길이 단위는 부모 요소나 뷰포트 크기에 따라 크기가 변한다
  • 반응형 디자인에 유용하며, 다양한 화면 크기에 적응할 수 있다
  • %: 부모 요소의 크기에 대한 백분율로 계산됨
  • em: 현재 요소의 font-size 값에 비례한 크기 설정
  • rem: 루트 요소(html)의 font-size 값에 비례한 크기 설정
  • vw: 뷰포트 너비의 1%를 기준으로 함
  • vh: 뷰포트 높이의 1%를 기준으로 함
