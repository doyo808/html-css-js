# white-space: 자동 줄바꿈 설정

  • normal: 기본값, 자동 줄바꿈
  • nowrap: 줄바꿈 없음, 한 줄로 표시
  • pre: 공백과 줄바꿈을 그대로 유지
  • pre-wrap: 공백은 유지하되 자동 줄바꿈
  • pre-line: 공백은 무시하고 줄바꿈은 유지

overflow: 요소의 범위를 벗어나는 내용에 대한 설정

  • scroll: 넘치는 내용이 스크롤 가능
