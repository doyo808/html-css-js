# text-shadow : 텍스트에 그림자를 넣는 속성(property)

  • 텍스트에 그림자를 추가하여 입체감을 줄 수 있다
  • text-shadow: [수평 거리] [수직 거리] [흐림 반경] [색상]
  • 예시: text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
  • , 를 사용해 연속적으로 여러 옵션들을 겹쳐 사용할 수 있다
