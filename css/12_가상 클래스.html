# Pseudo Class (가상 클래스)

  • 선택자 뒤에 추가로 사용하여 해당 요소의 특정 상황을 선택하는 것
  • 선택자 뒤에 :을 붙여 사용한다

# 가상 클래스의 종류

  • :hover - 마우스가 요소 위에 있을 때
  • :active - 요소가 활성화되었을 때 (클릭 중일 때)
  • :focus - 요소가 포커스를 받을 때 (입력란에 커서가 있을 때)
  • :first-child - 부모 요소의 첫 번째 자식 요소에 적용
  • :last-child - 부모 요소의 마지막 자식 요소에 적용
  • :nth-child(n/even/odd)
  • :not(selector)
  • :checked - 라디오박스나 체크박스가 체크되었을 때
  • :first-of-type
  • :last-of-type

# 링크에 적용되는 특정 가상 클래스 사용해보기

  • :link - 해당 요소가 방문한 적이 없는 링크 상태일 때
  • :visited - 방문한 적이 있는 상태일 때
  • :active - 해당 링크를 누르고 있는 상태일 때
  • :hover - 해당 요소에 마우스를 올린 상태일 때
