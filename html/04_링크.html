# 다른 웹 페이지로 이동할 수 있게 링크 걸기

- a태그 사용해 링크 걸기
- 자바스크립트로 링크 걸기

# a태그 사용하기

- a 태그의 href 속성에 원하는 url을 적어서 사용

- 자세한 내용이 궁금하다면 여기를 클릭하세요
- a 태그를 누르면 웹 브라우저의 주소창 안에 href 속성에 적힌 값을 입력하는 효과가 있다
- 구글가기1(google.com)
- 구글가기2(./google.com)
- 구글가기3(../google.com)
- 구글가기4(/google.com)
- 구글가기5(//google.com)
- 구글가기6(https://google.com)
- 요소와 속성 가기

# 웹에서 사용할 수 있는 상대 경로의 종류 (./, ../, /, //)

- file:///C:/javaweb_mhg/front-workspace/html/04_링크.html을 기준으로
- href="google.com"은 file:///C:/javaweb_mhg/front-workspace/html/google.com을 의미한다
- href="./google.com"은 file:///C:/javaweb_mhg/front-workspace/html/google.com을 의미한다
- href="../google.com"은 file:///C:/javaweb_mhg/front-workspace/google.com을 의미한다
- href="/google.com"은 file:///C:/google.com을 의미한다
- href="//google.com"은 file:///google.com을 의미한다

- https://comic.naver.com/webtoon/list?titleId=747269를 기준으로
- href="naver.com"은 https://comic.naver.com/webtoon/naver.com을 의미한다
- href="./naver.com"은 https://comic.naver.com/webtoon/naver.com을 의미한다
- href="../naver.com"은 https://comic.naver.com/naver.com을 의미한다
- href="/naver.com"은 https://comic.naver.com/naver.com을 의미한다
- href="//naver.com"은 https://naver.com을 의미한다
