# 웹페이지에 이미지 넣기

- image 태그를 사용해 웹 페이지에 원하는에 이미지를 넣을 수 있다
- src 속성: 출력하고자 하는 이미지의 경로
- alt 속성: 해당 이미지에 대한 설명 (이미지 출력 실패시 대신 출력되는 문구)
- alt 속성은 스크린 리더에 의해 읽혀지는 속성이기 때문에 해당 그림을 잘 표현할 수 있는 설명을 적어놓아야 한다.(웹 표준)
- width : 이미지의 너비
- height : 이미지의 높이 설정
- 너비와 높이는 하나만 설정하는 경우 비율이 유지한다 두 값 모두 설정하면 비율이 깨질 수 있다

# 로컬 이미지 사용하기

- 외부 사이트의 이미지 파일 경로를 사용하는 경우 해당 서버에 저장된 이미지를 매번 조회하는 것이기 때문에 남의 서버를 무단으로 사용하는것이 된다
- 이미지등의 정적 자원들은 자신의 서버에 보관하여 사용하는 것이 옳은 방법이다.
