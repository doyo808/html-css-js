# input

  • input 태그를 사용해 사용자로부터 다양한 입력을 받을 수 있다
  • input 태그의 type 속성에 따라 다양한 형태의 입력을 받을 수 있다
  • input 태그의 value 속성에 입력값을 지정할 수 있다
  • input 태그에 readonly 속성을 추가해놓으면 읽기 전용이 된다
  • input 태그에 disabled 속성을 추가해놓으면 비활성화된다
  • input 태그에 placeholder 속성을 추가해놓으면 입력란에 힌트가 나타난다
  • input 태그에 required 속성을 추가해놓으면 필수 입력란이 된다
  • input 태그에 autocomplete 속성을 추가해놓으면 자동완성 기능이 활성화된다
  • input 태그에 autofocus 속성을 추가해놓으면 페이지 로드 시 자동으로 포커스가 된다
  • input 태그에 form 속성을 추가해놓으면 해당 폼에 속하는 입력란이 된다
  • input 태그에 name 속성을 추가해놓으면 해당 입력란의 이름이 된다
  • input 태그에 tabindex 속성을 추가해놓으면 해당 입력란의 탭 순서를 지정할 수 있다
  • input 태그에 multiple 속성을 추가해놓으면 여러 개의 파일을 선택할 수 있다
  • input 태그에 accept 속성을 추가해놓으면 업로드할 수 있는 파일의 종류를 지정할 수 있다
타입 예시 설명
type="text" 일반적인 텍스트를 입력하는 곳
type="number" 숫자를 입력하는 곳 (숫자만 입력 가능)
type="password" 비밀번호를 입력하는 곳 (비밀번호를 가려준다)
type="checkbox" 관심 분야



같은 카테고리 내에서 여러가지를 선택할 수 있는 버튼
type="radio" 맛을 선택하세요!



같은 카테고리 내에서 하나만 선택할 수 있는 버튼
type="color" 색상을 선택할 수 있다
type="date" 날짜를 선택할 수 있다
type="file" 사용자의 컴퓨터에서 원하는 파일을 선택한다
type="hidden" 사용자에게는 보이지 않는 상태로 데이터를 보관하기 위해 사용한다
<textarea> 많은 양의 택스트를 입력하기 위해 사용하는 태그
<select>, <option> 주어진 항목들 중 하나를 선택할 수 있는 태그
