# Documnet Object Model (DOM)

  • 웹 브라우저가 HTML코드를 해석하여 해당 웹 페이지의 구조대로 생성해둔 객체
  • 자바스크립트의 documnet 객체를 통해서 해당 HTML요소에 접근할 수 있다

# 원하는 HTML 요소를 선택하기 위한 메서드들

  • document.getElementById(id) (겹치는 아이디가 있어도 처음거만 가져옴)
  • document.getElementsByTagName(tagName)
  • document.getElementsByClassName(className)
  • document.querySelector(cssSelector)
  • document.querySelectorAll(cssSelector)

# 선택한 HTML요소를 변경하기 위한 속성

  • element.innerHTML: 해당 요소의 내용을 변경한다 (태그 적용)
  • element.innerText: 해당 요소의 내용을 변경한다 (태그 미적용)
  • element.style: 스타일 속성을 변경한다
  • element.className: 클래스 속성을 변경한다
  • element.속성이름: 원하는 HTML속성을 변경한다
  • element.classList: 클래스 속성을 쉽게 변경할 수 있는 컬렉션을 사용한다

# 선택한 HTML요소로 활용할 수 있는 메서드

  • element.click(): 클릭한 효과
  • element.focus(): 포커스를 이동
  • element.blur(): 포커스 없앰
  • element.getAttribute(name): 속성 값 리턴
  • element.setAttribute(name, value): 속성 값 설정
  • element.removeAttribute: 속성 삭제
  • element.addEventListener(event, callback): 원하는 이벤트를 설정
